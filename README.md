# MyBrand Landing Page

A modern, responsive landing page built with clean HTML and CSS. Perfect for startups, portfolios, and product launches.

## 🎯 Overview

This project is a fully responsive landing page designed to showcase a modern brand. It features a sleek dark theme with cyan/blue accents, smooth scrolling navigation, and a mobile-friendly design that works seamlessly across all devices.

## ✨ Features

- **Responsive Design** - Layouts adapt beautifully across phones, tablets, and desktops
- **Modern Styling** - Bold gradients, clean spacing, and professional typography
- **Smooth Navigation** - Smooth scroll behavior and sticky header
- **Mobile Menu** - Hamburger menu for smaller screens
- **Dark Theme** - Eye-catching dark background with vibrant cyan accents
- **Feature Showcase** - Highlight key benefits and features
- **Pricing Plans** - Display different service tiers
- **Call-to-Action Buttons** - Encourage user engagement throughout

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- No build tools or dependencies required

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! No installation needed.

## 📖 Usage

Simply open the `index.html` file in your browser. The page includes:

- **Header** - Navigation menu with brand logo and CTA button
- **Hero Section** - Eye-catching introduction with key statistics
- **Features Section** - Three main feature cards highlighting benefits
- **Pricing Section** - Two pricing tiers (Free and Pro)
- **Footer** - Contact information and social media links

### Navigation
Click on any navigation link to smoothly scroll to different sections:
- Home
- Features
- Pricing
- Contact

### Mobile Menu
On smaller screens, click the hamburger menu (☰) to toggle navigation visibility.

## 🔧 Customization

### Change the Brand Name
In `index.html`, replace "MyBrand" with your brand name:
```html
<div class="logo"><span class="logo-mark">✦</span> YourBrand</div>
```

### Modify Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --accent: #22d3ee;      /* Change accent color */
    --bg: #0f172a;          /* Change background */
    --text: #f8fafc;        /* Change text color */
}
```

### Update Content
Simply edit the text in `index.html` to match your content:
- Headline, description, and buttons in hero section
- Feature cards
- Pricing plans
- Social media links

## 📱 Responsive Breakpoints
